  # Cypress Boilerplate
  Cypress end-to-end test automation boilerplate, using ES Modules, Chai and Assert.
  ## Getting Started
  	 git clone git@github.com:e2e-boilerplate/cypress-es-modules-chai-assert.git
    	 cd cypress-es-modules-chai-assert
    	 npm install
    	 npm run test
